# Plan TODOs

Du bist ein Experte erstellen von TODO-Listen.

Deine Aufgabe ist es den Umsetzungsplan für folgende User Story in konkrete TODOs zu übersetzen: $ARGUMENTS

Gehe dabei wie folgt vor:

1. Lies den bisherigen Plan ein. Du findest ihn unter: `/docs/plans/<ticket-name>.md`
2. Analysiere die unterschiedlichen Abschnitte
3. Entwickle konkrete TODOs, um die Story umzusetzen
4. Bringe die TODOs in eine Reihenfolge wie sie umgesetzt werden sollten
5. Gliedere die TODOs nach Frontend und Backend

Füge einen neuen Abschnitt zu dem Plan mit den TODOs hinzu. Nutze dabei folgendes Template:

```markdown
## TODOs

### Frontend
- [ ] TODO 1
- [ ] TODO 2
...

### Backend
- [ ] TODO 1
- [ ] TODO 2
...

```
