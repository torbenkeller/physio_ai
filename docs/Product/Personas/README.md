## Was ist eine Persona?

Personas sind fiktive, aber realistische Nutzerprofile, die typische Anwender repräsentieren. Sie helfen bei
Designentscheidungen und stellen sicher, dass das Produkt die tatsächlichen Bedürfnisse der Zielgruppe erfüllt.

Jede Persona enthält:

- Demografische Daten und Hintergrund
- Ziele und Motivationen
- Pain Points und Herausforderungen
- Technologie-Affinität
- Typische Nutzungsszenarien
