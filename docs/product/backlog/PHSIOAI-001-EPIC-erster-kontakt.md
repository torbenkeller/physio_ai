# Erster Kontakt

## Description
Dieses Epic umfasst alle Funktionalitäten für den ersten Kontakt mit einem neuen Patienten - vom ersten Anruf bis zur kompletten Terminplanung im System. Es stellt sicher, dass neue Patienten effizient in das PhysioAI System aufgenommen und ihre ersten Behandlungstermine geplant werden können.

## Business Value
- Optimierter Onboarding-Prozess für neue Patienten
- Reduzierte Fehlerquote bei der Patientenaufnahme
- Verbesserte Terminplanung und -verwaltung
- Grundlage für den gesamten Behandlungsprozess
- Professioneller erster Eindruck für neue Patienten

## User Stories
- [ ] [PHSIOAI-002](./PHSIOAI-002-STORY-patient-anlegen.md) Als Physiotherapeut möchte ich neue Patienten aus Namen und Geburtsdatum im System anlegen können, damit Behandlungstermine einem Patienten zugeordnet werden können
- [ ] [PHSIOAI-003](./PHSIOAI-003-STORY-behandlungen-pro-rezept-festlegen.md) Als Physiotherapeut möchte ich die Anzahl an tatsächlichen Behandlungsterminen pro Rezept für jeden Patienten eingeben können, damit ich für alte Patienten die Preise halten und für Neupatienten höhere Preise verlangen kann
- [ ] [PHSIOAI-004](./PHSIOAI-004-STORY-kalender-anzeigen.md) Als Physiotherapeut möchte ich den Kalender anzeigen können, damit ich mit dem Patienten am Telefon geeignete Behandlungszeiten abstimmen kann
- [ ] [PHSIOAI-005](./PHSIOAI-005-STORY-behandlungstermine-erstellen.md) Als Physiotherapeut möchte ich Behandlungstermine für einen Patienten im Kalender erstellen können, damit die vereinbarten Termine gespeichert und verwaltet werden
- [ ] [PHSIOAI-006](./PHSIOAI-006-STORY-termindauer-anpassen.md) Als Physiotherapeut möchte ich die Behandlungstermin-Dauer anpassen können, damit ich je nach Patientenbedürfnissen die passende Terminlänge festlegen kann
- [ ] [PHSIOAI-007](./PHSIOAI-007-STORY-patient-zuordnen.md) Als Physiotherapeut möchte ich einem Behandlungstermin einen Patienten zuordnen können, damit klar ist, wer behandelt wird
- [ ] [PHSIOAI-008](./PHSIOAI-008-STORY-terminbestaetigung-senden.md) Als Physiotherapeut möchte ich eine Terminbestätigung an den Patienten senden können, damit der Patient die vereinbarten Termine schriftlich bestätigt bekommt
