# Behandlungstermine erstellen

## User Story
Als Physiotherapeut möchte ich Behandlungstermine für einen Patienten im Kalender erstellen können, damit die vereinbarten Termine gespeichert und verwaltet werden.

## Beschreibung
Diese User Story ermöglicht es dem Physiotherapeuten, neue Behandlungstermine direkt im System zu erstellen und einem bestimmten Patienten zuzuordnen. Dies ist ein wesentlicher Bestandteil der Terminkoordination und stellt sicher, dass alle vereinbarten Termine strukturiert im System erfasst werden. 

Die Funktion soll intuitiv bedienbar sein und während des Telefongesprächs mit dem Patienten verwendet werden können. Jeder erstellte Behandlungstermin wird automatisch mit den entsprechenden Patientenstammdaten verknüpft und im Kalender visualisiert.

Nach der Erstellung des Termins soll dieser sofort im Kalender sichtbar sein und für weitere Verwaltungsaufgaben (wie Terminbestätigung oder Rezeptzuordnung) verfügbar werden.

## Akzeptanzkriterien
- [ ] Ich kann einen neuen Behandlungstermin mit Datum und Uhrzeit im Kalender erstellen
- [ ] Ich kann dem Behandlungstermin einen Patienten aus der bestehenden Patientenliste zuordnen
- [ ] Der erstellte Termin wird sofort im Kalender angezeigt und ist als belegt markiert
- [ ] Ich kann eine Standarddauer von 90 Minuten für den Behandlungstermin verwenden
- [ ] Ich kann mehrere Termine zur gleichen Zeit erstellen, falls notwendig (System gibt mir volle Kontrolle über die Terminplanung)
