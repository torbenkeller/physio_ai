# Abrechnungstermine PDF exportieren

## User Story
Als Physiotherapeut möchte ich generierte Abrechnungstermine als PDF exportieren können, damit ich sie ausdrucken und physisch an das Rezept anheften kann.

## Beschreibung
Nachdem Abrechnungstermine für ein Rezept generiert wurden, muss der Physiotherapeut diese als PDF-Dokument exportieren können. Das PDF enthält eine standardisierte Liste der berechneten Abrechnungstermine. Dieses Dokument wird ausgedruckt und physisch an das Papierrezept angeheftet, um als Nachweis für die Behandlung zu dienen.

Das PDF-Dokument enthält ausschließlich die Liste der berechneten Abrechnungstermine als Datumswerte.

## Akzeptanzkriterien
- [ ] PDF-Export-Button ist bei generierten Abrechnungsterminen verfügbar
- [ ] PDF enthält ausschließlich die Liste der generierten Abrechnungstermine als Datumswerte
- [ ] Abrechnungstermine sind in chronologischer Reihenfolge aufgelistet
- [ ] PDF wird automatisch heruntergeladen oder in neuem Tab geöffnet
- [ ] PDF-Dateiname ist eindeutig identifizierbar
