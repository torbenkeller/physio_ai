# Rechnung PDF herunterladen

## User Story
Als Physiotherapeut möchte ich eine generierte Rechnung als PDF herunterladen können, damit ich sie drucken und an den Patienten versenden kann.

## Beschreibung
Diese User Story ermöglicht es Physiotherapeuten, bereits generierte Rechnungen im PDF-Format herunterzuladen. Das PDF soll professionell formatiert sein und alle relevanten Rechnungsinformationen enthalten, einschließlich Praxisdaten, Patientendaten, Behandlungsdetails, Rechnungspositionen basierend auf der Gebührenordnung und Abrechnungstermine. Die heruntergeladene Rechnung kann sowohl digital versandt als auch ausgedruckt und physisch an den Patienten übermittelt werden.

## Akzeptanzkriterien
- [ ] Physiotherapeut kann über die Rechnungsübersicht eine Download-Schaltfläche für jede generierte Rechnung sehen
- [ ] Beim Klick auf Download wird automatisch ein PDF mit der Rechnung generiert und heruntergeladen
- [ ] Das PDF enthält alle erforderlichen Rechnungsinformationen: Praxisdaten, Patientenstammdaten, Rechnungsnummer, Behandlungsdetails mit Datum und Leistungsart
- [ ] Die Rechnungspositionen werden korrekt basierend auf der Gebührenordnung berechnet und angezeigt
- [ ] Die generierten Abrechnungstermine werden in der Rechnung aufgeführt
- [ ] Das PDF ist professionell formatiert und druckgeeignet
