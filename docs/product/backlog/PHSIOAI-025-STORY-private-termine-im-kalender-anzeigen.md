# Private Termine im Kalender anzeigen

## User Story
Als Physiotherapeut möchte ich private Kalendertermine im Kalender sehen, damit ich potentielle Überschneidungen von Behandlungsterminen mit meinen privaten Aktivitäten identifizieren kann.

## Beschreibung
Der Physiotherapeut benötigt die Integration privater Kalendertermine als Information im Kalender, um während der Terminplanung über potentielle Überschneidungen informiert zu sein. Wichtig ist dabei, dass private Termine die Buchung von Behandlungsterminen nicht verhindern - der Physiotherapeut behält die volle Entscheidungsfreiheit.

Private Termine sollen als Information sichtbar sein, aber visuell von Behandlungsterminen unterscheidbar bleiben. Dies ermöglicht eine bewusste Entscheidung bei Überschneidungen, ohne die Flexibilität der Terminbuchung einzuschränken.

Diese Funktionalität erweitert die Kalenderansicht um wichtige Kontextinformationen für eine optimale Terminplanung.

## Akzeptanzkriterien
- [ ] Private Kalendertermine werden als Information im Kalender angezeigt
- [ ] Private und Behandlungstermine sind visuell unterscheidbar (z.B. durch verschiedene Farben oder Markierungen)
- [ ] Private Termine verhindern nicht die Buchung von Behandlungsterminen
- [ ] Private Termine zeigen grundlegende Informationen (Titel, Zeit)
- [ ] Die Integration funktioniert mit gängigen Kalendersystemen
- [ ] Bei Überschneidungen bleibt die Entscheidungsfreiheit beim Physiotherapeuten
