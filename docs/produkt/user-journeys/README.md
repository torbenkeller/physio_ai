## Was ist eine User Journey?

User Journeys beschreiben den kompletten Ablauf einer Nutzerinteraktion mit dem System. Sie zeigen:

- Alle Schritte, die ein Nutzer durchläuft
- Emotionale Höhen und Tiefen (Satisfaction-Score)
- Touchpoints mit dem System
- Potenzielle Verbesserungsmöglichkeiten

Die Journeys sind als Mermaid-Diagramme visualisiert und dienen als Grundlage für User Story Maps.
